# Project Overview
The application is built using Next.js and includes:
- User registration and login (authentication)
- Protected pages accessible only after login
- CRUD functionality for items
- Backend API connected to a database

# Technologies Used
- Next.js
- Prisma ORM
- SQLite (local database)
- JWT Authentication
- JavaScript / CSS

# Features
- User authentication (register & login)
- Create, read, update, and delete items
- Protected routes for authenticated users
- Database integration using Prisma
